# FAWW Photobooth 📸

FAWW is a simple browser-based photobooth that captures images using the webcam and generates a vintage-style photo strip.

This project was built while learning web development and experimenting with the JavaScript Webcam API and Canvas.

## Features
- Webcam photo capture
- Countdown timer
- Photo strip generator
- Basic photo filters
- Download generated photostrip

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP

## How It Works
The application accesses the user's webcam through the browser, captures multiple images with a countdown timer, and combines them into a vertical photostrip using the HTML Canvas API.

The images can then be downloaded as a single photostrip.

## Purpose
This project was created as a learning experiment to understand browser APIs, image processing with canvas, and basic frontend–backend interaction.
